A lightbox gutenberg editor plugin to add beautiful image gallery.
 
== Description ==
 
This plugin will enable users to insert lightbox in beautiful image gallery in their posts, pages and CPT using Gu-
tenberg editor.


== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload `gutenberg-gallery-lightbox.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
